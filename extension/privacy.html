Comment Finder Privacy Policy

Effective date: July 11, 2026

Before publishing, the extension owner must replace the contact placeholder below and host this same policy at a public URL for the Chrome Web Store listing.

What Comment Finder does

Comment Finder is a browser extension that searches public comments for the video currently open in a YouTube tab. It uses YouTube API Services and does not scrape YouTube pages or access a YouTube account.

Information processed

When you submit a search, the extension sends the current video ID and your keyword to the extension operator's proxy. The proxy sends those values to the YouTube Data API to retrieve matching public comments and video metadata. The proxy receives an IP address from Cloudflare for short-window rate limiting, but does not persist the IP address, keyword, video ID, comment text, or author identity in its application telemetry.

Search results are held only in the open popup's memory. Comment Finder does not maintain an offline comment cache, profile users, use advertising technology, or sell or share this information for advertising. Aggregate proxy telemetry records only the endpoint called and its response status so the operator can monitor API quota use.

Device storage

The extension stores one device-local value in Chrome storage: the version of this Privacy Policy you accepted. It does not store comments, searches, or video history. Removing the extension clears this local value.

Service providers and sharing

Requests are processed by Cloudflare Workers and the YouTube Data API. Those services may process request data under their own terms and privacy policies. Comment Finder does not intentionally disclose search data to other parties.

Your choices

You can stop all further processing by not submitting a search or by removing the extension. Because Comment Finder does not create accounts or persist search/comment data, there is no remote profile to delete. For questions about this policy, contact: REPLACE_WITH_PRIVACY_CONTACT.

Terms and external policies

By using Comment Finder, you agree to the YouTube Terms of Service. Comment Finder's use of information received from YouTube API Services adheres to the YouTube API Services Terms of Service and the Google Privacy Policy.

Changes

If this policy changes, Comment Finder will require acceptance of the new policy version before its search feature is available again.
