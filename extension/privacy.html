Comment Finder Privacy Policy

Effective date: July 12, 2026

Before publishing, the extension owner must replace the contact placeholder below and host this same policy at a public URL for the Chrome Web Store listing.

What Comment Finder does

Comment Finder is a browser extension that searches public comments for the video or channel currently open in a YouTube tab. It uses YouTube API Services and does not scrape YouTube pages or access a YouTube account.

Information processed

Comment Finder reads the URL of the active tab when you open its popup or side panel, and while the side panel stays open it may re-read the active tab when you switch tabs or navigate on YouTube, so the panel can show the correct video or channel. URL access is limited to YouTube pages the extension is allowed to see (and the tab you invoke the extension on). The toolbar icon (active on supported YouTube watch, Shorts, channel, and handle pages, inactive elsewhere) is swapped by the browser from a declarative URL match, so the extension does not read tab URLs to change its icon. The URL stays on your device and is not sent to the proxy except as the video ID, channel ID, or handle needed for a search you submit.

When you submit a search, the extension sends the current video ID or channel ID (resolved from a channel URL or @handle when needed) and your keyword to the extension operator's proxy. The proxy sends those values to the YouTube Data API to retrieve matching public comments and video or channel metadata. For channel searches, the proxy may also request titles for the videos that contain matching comments. The proxy receives an IP address from Cloudflare for short-window rate limiting, but does not persist the IP address, keyword, video ID, channel ID, handle, comment text, or author identity in its application telemetry.

Search results stay on your device for the current browser session so the popup or side panel can restore them when reopened on the same video or channel. Comment Finder does not maintain a permanent offline comment cache, profile users, use advertising technology, or sell or share this information for advertising. Aggregate proxy telemetry records only the endpoint called and its response status so the operator can monitor API quota use.

Device storage

The extension stores the version of this Privacy Policy you accepted in Chrome local storage. It also keeps recent per-video, per-channel, and per-handle popup state (keyword, metadata, matching comments, source-video titles for channel results, and pagination position) in Chrome session storage for the current browser session. Session data is cleared when the browser closes. Removing the extension clears both values.

Service providers and sharing

Requests are processed by Cloudflare Workers and the YouTube Data API. Those services may process request data under their own terms and privacy policies. Comment Finder does not intentionally disclose search data to other parties.

Your choices

You can stop all further processing by not submitting a search or by removing the extension. Closing the browser clears session search state. Terms and external policies
