# Extension assets

## Toolbar icons

`icon-active-*.png` and `icon-inactive-*.png` are Comment Finder’s action icons (16/32/48/128). The active set uses the brand red (`#F70130`); the inactive set is the same artwork in gray. The background service worker shows the active icon on supported YouTube watch, Shorts, `/channel/…`, and `@handle` tabs, and the inactive icon elsewhere. `icon-active.png` / `icon-inactive.png` are 512px masters for regeneration and store listing art.

## Attribution assets

`developed-with-youtube.png` is the unmodified dark, sentence-case "Developed with YouTube" PNG supplied from the official YouTube API branding-guidelines download link:

https://developers.google.com/static/youtube/images/developed-with-youtube-sentence-case-dark.png

`developed-with-youtube-light.png` is the unmodified light variant for dark UI backgrounds:

https://developers.google.com/static/youtube/images/developed-with-youtube-sentence-case-light.png

Both assets must stay linked to YouTube content when displayed and must not be altered.
